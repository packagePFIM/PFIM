# evaluate the derivatives of the model error.

evaluate the derivatives of the model error.

## Arguments

- modelError:

  An object `ModelError` that defines the model error.

- evaluationModel:

  A dataframe giving the outputs for the model evaluation.

## Value

The matrices sigmaDerivatives and errorVariance.
