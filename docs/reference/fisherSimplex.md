# Compute the fisher.simplex

Compute the fisher.simplex

## Arguments

- simplex:

  A list giving the parameters of the simplex.

- optimizationObject:

  An object `Optimization`.

- outcomes:

  A vector giving the outcomes of the arms.

## Value

A list giving the results of the optimization.
