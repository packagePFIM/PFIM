# Extract arm data for reporting

Extracts arm-level design information (name, size, outcomes, doses,
sampling times) formatted for inclusion in HTML reports.

## Usage

``` r
getArmData(arm, ...)
```

## Arguments

- arm:

  An object of class `Arm`.

- ...:

  Additional arguments.

## Value

A list of named lists, one per sampling outcome.
