# Evaluation of a clinical design

Evaluation of a clinical design

## Arguments

- design:

  An object `Design` to evaluate.

- model:

  An object `Model` used for evaluation.

- fim:

  An object `Fim` to store results.

## Value

The `Design` object with evaluated arms and aggregated global FIM.
