# plotSensitivityIndices

Generates sensitivity index plots (partial derivatives of model
responses with respect to population parameters).

## Usage

``` r
plotSensitivityIndices(pfimproject, ...)
```

## Arguments

- pfimproject:

  An object of class `PFIMProject`.

- ...:

  Additional arguments.

## Value

A named list of sensitivity index plots per design.
