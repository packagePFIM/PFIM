# : replace variable in the LibraryOfModels

A utility function designed to rename or replace variable names within
model strings (e.g., library equations). It ensures safe replacement by
protecting reserved mathematical terms and keywords.

## Usage

``` r
replaceVariablesLibraryOfModels(text, old, new)
```

## Arguments

- text:

  the text

- old:

  old string

- new:

  new string

## Value

text with new string
