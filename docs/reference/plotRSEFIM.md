# Bar plot for Relative Standard Errors (RSE)

Bar plot for Relative Standard Errors (RSE)

bar plot for the RSE.

Plot Relative Standard Errors (RSE%) for Population FIM

## Arguments

- fim:

  An object of class
  [`PopulationFim`](https://packagePFIM.github.io/PFIM/reference/PopulationFim.md)
  containing the calculated RSE values.

- evaluation:

  An object of class `Evaluation` providing the context and parameter
  names for the plot.

## Value

A ggplot object representing the RSE bar plot.

The bar plot of the RSE.

A plot object (typically `ggplot2` or `lattice`) displaying the RSE% for
structural and variance parameters.
