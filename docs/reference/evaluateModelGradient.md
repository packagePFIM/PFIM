# evaluate the gradient of the model

Computes the numerical gradient of the model response with respect to
the structural parameters using the finite difference method.

## Arguments

- model:

  An object `Model` that defines the model.

- arm:

  A object `Arm` giving the arm

## Value

A data frame that contains the gradient of the model.
