# run

This function performs the evaluation or the optimization of a
experimental design based on the settings provided in a `PFIMProject`
object.

## Usage

``` r
run(pfimproject, ...)
```

## Arguments

- pfimproject:

  An object of class `PFIMProject` containing the project settings,
  model definitions, and design parameters.

- ...:

  Additional arguments.
