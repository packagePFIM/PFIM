# Dcriterion

Computes the D-criterion of the Fisher Information Matrix. The
D-criterion is calculated as the determinant of the FIM raised to the
power of 1 over the number of parameters.

## Arguments

- fim:

  An object of class `Fim`.

## Value

A double giving the D-criterion of the Fim.
