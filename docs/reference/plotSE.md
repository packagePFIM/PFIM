# plotSE

Bar plot of Standard Errors (SE) for fixed effects and variance
components.

## Usage

``` r
plotSE(pfimproject, ...)
```

## Arguments

- pfimproject:

  An object of class `PFIMProject`.

- ...:

  Additional arguments.

## Value

A bar plot of SE values.
