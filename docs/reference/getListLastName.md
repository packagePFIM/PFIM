# getListLastName: Get names of the deepest elements in a nested list

Recursively traverses a nested list to extract the names of the elements
at the lowest level of the hierarchy.

## Usage

``` r
getListLastName(list)
```

## Arguments

- list:

  A `list` (potentially nested).

## Value

A `character` vector containing the names of the last elements.
